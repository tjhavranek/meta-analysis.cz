## Coverage

All words, punctuation, diacritics, emoji, and non-whitespace line/paragraph boundaries match; the four whitespace-only deviations are listed above.

Row accounting is **26 CSV rows → 25 JSON posts**. Every CSV `ShareCommentary` is nonempty. The sole dropped row is the correctly excluded row dated **2024-09-02 15:16:49**; no other row was dropped, and no JSON post was invented. All 25 `date` values equal the first ten characters of the source `Date`, all 25 `datetime` values equal the complete source `Date`, and all 25 `url` values equal the corresponding source `ShareLink` verbatim.

The published page contains **25** post articles. Each JSON post appears once with its complete JSON text, a nonempty article `id` unique across the page, a matching self-anchor, and the correct verbatim original-post link. The page references **15** images: all 15 files exist in `social-img/`, all 15 references agree with the corresponding JSON `images` lists, no image is used by two posts, and no file in `social-img/` is left unused. None of the prohibited interface-furniture strings, follower counts, or reaction tallies survives in a post. Ordinary authored prose containing lowercase words such as “comments” or “like” was not treated as LinkedIn interface furniture because it is present verbatim in the authoritative CSV.
